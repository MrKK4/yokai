## 2024-05-24 - Accessibility on Icon-only buttons\n**Learning:** Compose `Icon` components inside `IconButton`s often have `contentDescription = null` which hides them from screen readers. It's better to provide a contentDescription for icon-only buttons.\n**Action:** Add `contentDescription` using Moko resources (`dev.icerock.moko.resources.compose.stringResource(MR.strings.x)`) for accessibility.
