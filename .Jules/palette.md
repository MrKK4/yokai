## 2024-05-09 - Missing ARIA labels in Jetpack Compose UI
**Learning:** Found multiple instances of `contentDescription = null` in Jetpack Compose UI elements (`app/src/main/java/yokai/presentation/extension/repo/component/ExtensionRepoItem.kt`), specifically for interactive icon buttons like delete, and informative icons like add/check. Setting content description to null on an `IconButton` or interactive icon violates accessibility best practices, making the interface inaccessible for screen reader users.
**Action:** When updating Jetpack Compose UI elements, always provide meaningful strings from `MR.strings` (Moko Resources) for the `contentDescription` of interactive elements (like `IconButton`s) and informative icons. Ensure appropriate localized strings are used (e.g., `stringResource(MR.strings.delete)`, `stringResource(MR.strings.add)`).
