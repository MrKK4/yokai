## 2026-05-23 - Missing contentDescriptions on IconButtons\n**Learning:** IconButtons containing purely informational/interactive Icons in ExtensionRepoItem.kt were initially implemented with `contentDescription = null`, breaking accessibility for screen reader users.\n**Action:** When adding or reviewing `IconButton` elements in Jetpack Compose, always ensure the inner `Icon` is provided a meaningful `contentDescription` using a localized string resource (e.g., via `mokoStringResource(MR.strings.x)`) rather than `null`.\n
